# 第4回
## 逃げろこうかとん（ex04/dodge_bomb.py）
### ゲーム概要
- ex04/dodge_bomb.pyを実行すると，1600x900のスクリーンに草原が描画され，こうかとん
を移動させ飛び回る爆弾から逃げるゲーム
- こうかとんが爆弾と接触するとゲームオーバーになる
### 操作方法
- 矢印キーでこうかとんを上下左右に移動する
### 追加機能
- 爆弾を個数を１つ追加
- 爆弾の画像に入れ替え（figファイル内にbomb.pngが必要）
- 着弾時に文字の描画，爆弾の停止，こうかとんの画像の切り替え
